# nira_connect_mobile_app
NIRA Connect is an Android mobile application developed using Kotlin and SQLite that enables Ugandan citizens to apply for National ID registration, renewal, and replacement through their smartphones. This academic project was developed as part of the NW 201 Mobile Application Development course at Uganda Technology and Management University.

 Features
Core Functionalities
User Authentication: Secure registration and login system

ID Services: Registration, renewal, and replacement applications

Application Tracking: Real-time status monitoring with unique tracking numbers

Dashboard Analytics: Interactive charts and statistics visualization

Offline Operation: Complete functionality without internet using SQLite

Technical Features
Database: SQLite with 185 sample applications for testing

Charts: MPAndroidChart integration for data visualization

UI/UX: Material Design principles with intuitive navigation

Security: Input validation and secure data storage

Technology Stack
Language: Kotlin 100%

Minimum SDK: Android 7.0 (API 24)

Database: SQLite

Charts: MPAndroidChart v3.1.0

UI Framework: Android XML with Material Design

Development Tool: Android Studio
